Introduction

Th 8/19. Brief meeting for syllabus review, SN 014

Week 1: Setup & Basic HTML

Tu 8/24. Lecture/discussion, Zoom. Read chapters 1-4 in Duckett, HTML & CSS. (Don't worry, it's mostly images, and also don't worry about following along with the "creating a web page" sections, because we'll do that stuff together in VSCode or your preferred text editor during class.) Set up VSCode or another text editor before class: details at https://webdemos.web.unc.edu/setup-with-vscode/ and https://webdemos.web.unc.edu/code-editors-and-ides/. Also recommended: Introduction, Basic Web Pages, and Links & Images sections at https://www.internetingishard.com/html-and-css/ (Note that these tutorials reference the Atom text editor, not VSCode. The two applications work in very similar ways for this kind of project, so you're welcome to use either, or whichever alternative you prefer.)

Th 8/26. In-person work session, demos, examples. Focus on getting everyone's Assignment 1 project directory set up so you can start adding content. Maybe a bit of CSS preview before next week. In SN 014, masked, or join via Zoom.

Week 2: Introduction to CSS: Basic Styling, the Box Model, Display

Tu 8/3. Zoom. Lecture. Read Interneting is Hard: "Hello, CSS"; "The Box Model"; "CSS Selectors" before class. (Don't worry about the specifics of the setup they walk you through; stick to the model we've set up for A1 for now. Just get the basics of CSS syntax, box model, and display properties for now.) Highly recommended: CSS Guidelines, especially "Syntax & Formatting" & "Naming Conventions".
