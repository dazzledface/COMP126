COMP126: Practical Web Design & Development for Everyone

Course Schedule, Fall 2021

info & policies

To note:

This schedule is subject to change. Please resist the impulse to print out and rely on a hard copy!

Most Tuesdays at 12:30: full-class lecture/discussion meetings on Zoom; see Piazza for link

Most Thursdays at 12:30: Q&A/demo session in Sitterson Hall room 014, or Zoom in; see Piazza for link

All Zoom class meetings will be recorded and made available to you on Sakai under Class Recordings

Introduction

Th 8/19. Brief meeting for syllabus review, SN 014.

Week 1: Setup & Basic HTML

Tu 8/24. Lecture/discussion, Zoom. Read chapters 1-4 in Duckett, HTML & CSS. (Don't worry, it's mostly images, and also don't worry about following along with the "creating a web page" sections, because we'll do that stuff together in VSCode or your preferred text editor during class.) Set up VSCode or another text editor before class: details at https://comp126.com/setup.html. Also recommended: Introduction, Basic Web Pages, and Links & Images sections at https://www.internetingishard.com/html-and-css/ (Note that these tutorials reference the Atom text editor, not VSCode. The two applications work in very similar ways for this kind of project, so you're welcome to use either, or whichever alternative you prefer.)

Th 8/26. In-person work session, demos, examples. Focus on getting everyone's Assignment 1 project directory set up so you can start adding content. Maybe a bit of CSS preview before next week. In SN 014, masked, or join via Zoom.

Week 2: Introduction to CSS: Basic Styling, the Box Model, Display

Tu 8/3. Zoom. Lecture. Read Interneting is Hard: "Hello, CSS"; "The Box Model"; "CSS Selectors" before class. (Don't worry about the specifics of the setup they walk you through; stick to the model we've set up for A1. Just get the basics of CSS syntax, box model, and display properties for now.) Highly recommended: CSS Guidelines, especially "Syntax & Formatting" & "Naming Conventions".

Th 9/2. SN 014. Work session, demos, examples. Sakai Quiz 1 @ 12:30pm.

Week 3: More CSS: Layout, Positioning, Floats

Tu 9/7. Zoom class. Read Interneting is Hard: "Floats" and "Advanced Positioning" before class. (You can skip the Flexbox section for now.)

Th 9/9. SN 014. Work session, demos, examples.

Su 9/12. Assignment 1 Due, 11:55pm EST.

Week 4: Introduction to Responsive Design & HTML5 Markup

Tu 9/14. No synchronous class meeting. Before class on Thursday 9/16: 1) read Frain, Responsive Web Design with HTML5 & CSS3 (the course textbook) ch. 1-3; and 2) watch The Beginner's Guide to Responsive Web Design.

Th 9/16. Zoom-only lecture/Q&A. No in-person meeting, although you're welcome to Zoom in from SN014 if you wish.

Week 5: More Responsive Design & Media Queries

Tu 9/21. Sakai Quiz 2 @ 12:30pm (Sakai > Tests & Quizzes) on Frain ch 1-3, then lecture/Q&A. All on Zoom/Sakai.

Th 9/23. Class in SN 014 w/Zoom option. Q&A, demos, lectures, as needed.

Su 9/26. Assignment 2 Due, 11:55pm EST.

Week 6: Fluid Layouts & Flexbox

Tu 9/28. Zoom. Lecture. Read Frain ch. 4 before class.

Th 9/30. SN 014. Work session, demos, examples. Sakai Quiz 3 @ 12:30pm.

Week 7: CSS Grid

Tu 10/5. Zoom. Lecture. Read Frain ch. 5 before class.

Th 10/7. SN 014/Zoom. Work session, demos, examples.

Week 8: Pause

Tu 10/12 & Th 10/14. No new material.

From Tessa: In addition to my regular Zoom office hours (TuTh 11am-12:15pm EST at my Zoom OH link), I will offer optional Q&A/review/discussion sessions on Zoom during both of this week's class times (12:30-1:45pm EST both days at our usual class time Zoom link). Technically, of course, all classes and office hours are cancelled on Tuesday, but I'm planning to remain available during those times on both Tuesday and Thursday, just in case anyone could use a little extra help or just wants to keep busy. There is absolutely no pressure to show up; I only want you to know that my support is there, regardless of what you need it for.

So join in as much or as little as you like with questions, to go on CSS and design tangents, and/or just to chat. We'll keep it very light and hopefully enjoyable. I won't be lecturing, and I won't introduce any crucial new material (or if I do, I'll make sure to present it again to the whole class when we reconvene).

Please use this time however you need: to get caught up, explore new things, or simply rest, recoup, reflect, and, I hope, begin to heal.

Please note: Various ULA office hours may be rescheduled or cancelled this week. Updates and changes will be posted on Piazza.
